##  Docker should be installed on your machine

#### For run this app, clone the repo to your machine then use 'docker-compose up' command
script, flask app will run automatically, you can run script again if you want to change data on db
#### Urls 
 - host:5000/all - Lists all data from a collection
 - host:5000/person/id - Get one specified entity
 - host:5000/delete/id - Delete one specified entity
